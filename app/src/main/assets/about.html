Project by: Jacob Laney 李傑

Structural Features

  • I implemented swipe navigation for the app. Three pages exist: a pm2.5 sensor page, data history page, and an about page.
  • I added buttons to the swipe navigation feature to allow easier control depending on the device that is used.
  • The pm2.5 sensor page now has a moving graph that shows the data updating live.

Bluetooth Features

  • A lost connection to the device will return the user to the bluetooth connection page.
  • A thread for the Bluetooth connection routine. User experience is no longer interrupted by this slow task.
  • A timer task that updates the device list every few seconds. Now the user does not need to hit a button to refresh the list.

Data Storage Features

  • Began implementing data storage, but this feature is only partially complete.
  • The data history page shows some test data on a graph. This is a proof of concept for my data storage class "PMBTFileService".
